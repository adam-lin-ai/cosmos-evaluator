# Cosmos Evaluator

The Cosmos Evaluator is a platform for automatically checking the augmented output of a Cosmos model, and evaluating whether the augmentation was successful, filtering out bad results.

## Getting Started

To get started with the Cosmos Evaluator, and for more information about what is offered, please visit the documentation in the [Getting Started Guide](docs/getting-started.md).

Other documentation for this repository may be found under the docs/ directory.
